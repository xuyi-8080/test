GIT初始化：
 设置用户名：git --global user.name 'xxxxx'
 设置邮箱  ：git --global user.email 'xxxx@xx.com'
 
 mkdir name     创建一个文件夹
 cd  name         进入一个文件夹
 pwd                显示当前所处地址
 git init            设置当前这个文件夹为库  并生成一个隐藏文件夹 里面有库的设置
 
向库里添加文件
 touch  xx.c  添加一个文件到文件夹
 git add xx.c 添加文件到暂存区
 git commit -m '文字描述' 将暂存区的文件提交到库

修改库里的文件
 vi name  编辑某个文件
注: 按i进入insert模式, 编辑完成后按esc退回到命令行模式 输入:wq保存修改并退出vi
 git add xx.c 将修改的文件添加到暂存区
 git commit -m '文字描述'

删除库里的文件
 rm -rf name   删除文件
 git rm 文件名  向暂存区中填入删除该文件的命令
 commit -m '文字描述'  将更改提交到库中进行同步

下载别人的库
git clone 地址

将本地的库上传到远程库中
git push
